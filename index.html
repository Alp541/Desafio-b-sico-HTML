HTML hiper text markup linguage.

O html é uma linguagem de marcação utilizada para o desenvolvimento de páginas web , sendo assim não é consoderado uma linguagem de programação. Podemos tomar como base que o html é um tipo de documentação a qual os browser podem interpretar e repasar um resultado o qual seria nossas páginas webs.

Tags


As tags do HTML são usadas para a estruturação do conteúdo da documentação , exeplo para formatar um texto ou para adicionarmos imagens e outros atributos. As tags HTML básicas são essenciais para criar qualquer documento HTML. Sem elas, um documento HTML não teria nenhum conteúdo e não seria exibido corretamente. explos simples de tags :
